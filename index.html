Rexko

A tool for making dictionaries

Download Download View Rexko on GitHub

Rexko is a project to develop a tool to create and publish dictionary data. It has been in use for the past few years to create the entries for the Latin dictionary at ttt.frath.net; recently we have been working on transitioning the application from an internal tool to one suitable for public use.
